## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-initialize`
- `allow-can-request-ads`
- `allow-get-consent-status`
- `allow-load-banner`
- `allow-load-interstitial`
- `allow-request-consent`
